# Getting Started

_Quick start guide to get up and running with Feathers component library in your React application._

## Installation

Follow these steps to integrate Feathers into your React project with Tailwind CSS.

### 1. Install Feathers

<Tabs defaultValue="npm" className="my-6">

  <TabsContent value="pnpm">

  </TabsContent>
  <TabsContent value="bun">
    ```bash
    bun add @future-house/feathers
    ```
  </TabsContent>
</Tabs>

> If you want to use feathers prebuilt theme, without customizing it, please import feathers styles in your CSS file and skip to the 6th step `@import "../node_modules/@future-house/feathers/index.css";`

### 2. Install Tailwind CSS

If you don't have Tailwind CSS set up in your project, install it:

```bash
npm install tailwindcss @tailwindcss/postcss autoprefixer
```

### 3. Import Tailwind in your CSS

Add the `@import “tailwindcss”;` import to your main CSS file.

```javascript
@import "tailwindcss";
```

### 4. Add Feathers to your CSS as a source to be used by Tailwind

After installing Tailwind CSS, you need to inject feathers styles into your project.

- Open or create your main CSS file. This file is typically located at `src/index.css` or `src/styles.css`.
- Add the `@future-house/feathers` as a source to be used by Tailwind at the top of your CSS file:

```css
@source '../node_modules/@future-house/feathers';
```

```css
/* In your main CSS file (e.g., index.css or app.css) */
@import '@future-house/feathers/index.css';
```

### 6. Initialize Theme Provider

Wrap your application with the ThemeProvider to enable theming and dark mode support:

```jsx
import { ThemeProvider } from '@future-house/feathers';

function App() {
  return (
    <ThemeProvider defaultTheme="system" storageKey="feathers-theme">
      <div className="bg-background text-foreground min-h-screen">
        {/* Your app content */}
      </div>
    </ThemeProvider>
  );
}

export default App;
```

```jsx
import React from 'react';
import { Button } from '@future-house/feathers';

function App() {
  return (

      <Button onClick={() => alert('Hello from Feathers!')}>Click me</Button>
    </div>
  );
}

export default App;
```
