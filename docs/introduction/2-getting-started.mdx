# Getting Started

_Quick start guide to get up and running with Feathers component library in your React application._

## Setup

### 1. Install Tailwind CSS

Feathers is built with Tailwind CSS. If you don't have Tailwind CSS set up in your project, follow the [Tailwind CSS installation guide](https://tailwindcss.com/docs/installation).

### 2. Configure Tailwind CSS

Add Feathers to your `tailwind.config.js`:

```javascript
// tailwind.config.js
module.exports = {
  content: [
    './src/**/*.{js,jsx,ts,tsx}',
    './node_modules/@future-house/feathers/**/*.{js,jsx,ts,tsx}',
  ],
  theme: {
    extend: {},
  },
  plugins: [],
};
```

### 3. Import Styles

Import the Feathers CSS in your main CSS file or `index.css`:

```css
@import '@future-house/feathers/styles';
```

### Your First Component

Here's a simple example using the Button component:

```jsx
import React from 'react';
import { Button } from '@future-house/feathers';

function App() {
  return (
    <div className="p-8">
      <Button onClick={() => alert('Hello from Feathers!')}>Click me</Button>
    </div>
  );
}

export default App;
```

## Theme Setup

### Dark Mode Support

Feathers includes built-in dark mode support. Wrap your app with the ThemeProvider:

```jsx
import { ThemeProvider } from '@future-house/feathers';

function App() {
  return (
    <ThemeProvider defaultTheme="system" storageKey="my-app-theme">
      <div className="bg-background text-foreground min-h-screen">
        {/* Your app content */}
      </div>
    </ThemeProvider>
  );
}
```

### Theme Toggle

Add a theme toggle button to let users switch between light and dark modes:

```jsx
import { ThemeToggle } from '@future-house/feathers';

function Header() {
  return (
    <header className="flex items-center justify-between p-4">
      <h1>My App</h1>
      <ThemeToggle />
    </header>
  );
}
```
