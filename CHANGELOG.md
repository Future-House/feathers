## [1.0.0] - 8-21-2025

### Added

- Initial redesigned release of Feathers component library
- Core components built with React 19, TypeScript, Tailwind CSS, and Radix UI
- Storybook documentation for all components
- Jest unit tests with React Testing Library
- Tree-shaking support for optimal bundle sizes
- Dark mode support across all components
- Multiple import path patterns for flexible usage
- Custom build system with TypeScript declaration generation
- Example application with Vite, Tailwind, and TypeScript
