## Overview

The ThemeToggle component provides three different ways to switch between light and dark themes. Each variant offers a different user experience:

- **Button**: Simple toggle between light and dark modes
- **Switch**: Animated sliding toggle with theme icons on the thumb
- **Dropdown**: Full theme selector with light, dark, and system options

**_The `ThemeToggle` pages are wrapped with their own nested theme, so there may be some weirdness when toggling themes._**
