## Accessibility

The `ThemeToggle` component follows accessibility best practices:

- Provides proper ARIA labels for screen readers
- Maintains keyboard navigation support with Enter and Space keys
- Uses semantic button element for proper interaction
- Announces theme changes to assistive technologies
- Maintains proper focus management

## Best Practices

- Place theme toggle in a consistent location (like header or settings menu)
- Use the component within a `ThemeProvider` context for proper functionality
- Consider adding a tooltip or label to explain the theme toggle functionality
- Test the toggle with keyboard navigation to ensure accessibility
- Provide visual feedback when theme changes occur
- Consider placement and visibility in both light and dark themes
