## Accessibility

The CircularProgress component includes proper accessibility features:

- Uses `role="progressbar"` for screen reader compatibility
- Includes `aria-valuemin`, `aria-valuemax`, and `aria-valuenow` attributes
- The `aria-valuenow` attribute is omitted for indeterminate progress as recommended by ARIA guidelines

## Best Practices

- Use determinate mode when you can calculate progress percentage
- Use indeterminate mode for operations where duration is unknown
- Pair with descriptive text when possible
- Consider the context - smaller sizes for inline elements, larger for prominent loading states
- Choose appropriate thickness based on the overall design density
