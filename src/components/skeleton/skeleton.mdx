## Accessibility

The `Skeleton` component includes:

- Proper ARIA attributes for screen readers
- Semantic HTML structure that indicates loading state
- Sufficient color contrast for visibility
- Respects user preferences for reduced motion

## Best Practices

- Use skeleton placeholders that match the shape and size of your actual content
- Ensure skeleton layouts are consistent with your final content structure
- Keep skeleton animations subtle and not overwhelming
- Use skeletons for perceived performance improvement during loading states
- Consider showing skeleton placeholders for at least 300ms to avoid flashing
