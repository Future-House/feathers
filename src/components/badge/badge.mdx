## Accessibility

The `Badge` component:

- Uses semantic HTML with proper contrast ratios
- Maintains readability across different themes
- Supports screen readers with meaningful content
- Works with keyboard navigation when used in interactive contexts

## Best Practices

- Use badges sparingly to avoid visual clutter
- Choose appropriate variants based on the information hierarchy
- Ensure sufficient color contrast for accessibility
- Keep badge text concise and meaningful
- Use consistent badge styling throughout your application
- Consider the context when choosing between variants
- Combine with icons when additional context is helpful
