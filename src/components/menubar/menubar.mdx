## Accessibility

The `Menubar` component follows the [WAI-ARIA Menubar Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/menubar/) and includes:

- Full keyboard navigation with arrow keys, Enter, and Escape
- Proper ARIA attributes for screen readers
- Focus management and tab order
- Support for keyboard shortcuts display
- Automatic role and state announcements

## Best Practices

- Use consistent keyboard shortcuts and display them in menu items
- Group related functionality together using separators
- Keep menu depth reasonable (avoid deeply nested submenus)
- Provide clear, descriptive labels for all menu items
- Use icons consistently to help with visual recognition
- Consider the menu's position in your application layout
