## Accessibility

The `ToggleGroup` component follows the [WAI-ARIA Toolbar Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/toolbar/) and includes:

- Keyboard navigation with arrow keys, Home, and End
- Screen reader support with proper role announcements
- Focus management within the group
- Individual toggle accessibility with `aria-pressed` states
- Support for both single and multiple selection modes

## Best Practices

- Use single selection for mutually exclusive options
- Use multiple selection when users can choose several options
- Provide clear visual feedback for selected states
- Group related functionality together logically
- Ensure adequate spacing between toggle items
- Test with keyboard navigation and screen readers
- Consider the orientation (horizontal/vertical) based on available space
- Use meaningful icons and labels for better usability
