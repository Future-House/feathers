## Accessibility

The `RadioGroup` component follows the [WAI-ARIA Radio Group Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/radio/) and includes:

- Keyboard navigation with arrow keys and Tab
- Space and Enter key activation
- Screen reader support with proper ARIA roles
- Focus management within the group
- Required field validation support

## Best Practices

- Always provide labels for radio button items
- Use radio groups when only one option can be selected
- Group related options logically together
- Provide clear, descriptive labels that work without context
- Consider the visual layout (vertical vs horizontal) based on space
- Use default values when appropriate to guide user choices
