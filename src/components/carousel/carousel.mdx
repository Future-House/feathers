## Accessibility

The `Carousel` component follows accessibility best practices:

- Supports keyboard navigation with Tab, Enter, and arrow keys
- Provides proper ARIA labels and roles for screen readers
- Maintains focus management when navigating between items
- Includes proper semantic structure for carousel content
- Supports screen reader announcements for slide changes
- Allows users to pause auto-playing carousels

## Best Practices

- Provide clear navigation controls for users
- Use appropriate slide transitions and timing
- Ensure carousel content is accessible via keyboard
- Consider providing alternative navigation methods
- Test with assistive technologies for accessibility
- Use consistent spacing and sizing for carousel items
- Provide indicators when showing multiple slides
- Consider performance with large numbers of slides
