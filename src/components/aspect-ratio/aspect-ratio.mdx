## Accessibility

The `AspectRatio` component:

- Maintains semantic HTML structure
- Preserves accessibility attributes when using `asChild`
- Ensures proper focus management for interactive content
- Works with screen readers and keyboard navigation

## Best Practices

- Use consistent aspect ratios across your application for visual harmony
- Choose appropriate aspect ratios based on content type (16:9 for videos, 1:1 for avatars)
- Consider responsive design when choosing aspect ratios
- Use the `asChild` prop when you need to maintain the aspect ratio of interactive elements
- Test on different screen sizes to ensure content remains accessible
- Provide alternative text for images within aspect ratio containers
