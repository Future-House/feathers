## Accessibility

The `Resizable` components include:

- Keyboard support with arrow keys for resize operations
- Screen reader announcements for resize actions
- Focus management for resize handles
- ARIA attributes for handle identification
- Proper tab order through resizable interfaces

## Best Practices

- Set appropriate minimum and maximum sizes for panels
- Use visual handles to indicate resizable areas
- Consider the content needs when setting default sizes
- Test resize behavior at different viewport sizes
- Provide smooth resize animations for better user experience
- Save panel sizes in user preferences when appropriate
