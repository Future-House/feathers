## Accessibility

The `NavigationMenu` component follows the [WAI-ARIA Navigation Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/navigation/) and includes:

- Keyboard navigation with Tab, arrow keys, Enter, and Escape
- Screen reader support with proper ARIA attributes
- Focus management when navigating between items
- Support for both horizontal and vertical orientations
- Automatic viewport positioning to prevent content overflow

## Best Practices

- Use clear, descriptive link text that makes sense out of context
- Group related navigation items logically
- Consider the visual hierarchy when designing menu content
- Test with keyboard navigation to ensure smooth interactions
- Use appropriate delay timing based on your users' needs
- Provide alternative navigation methods for complex structures
